Understanding the Box Model

The CSS box model is a fundamental concept in web design that describes how elements are structured and displayed on a webpage.

About CSS

CSS (Cascading Style Sheets) controls the style and layout of web pages. It allows developers to separate structure (HTML) from presentation (CSS).

Box Model Components

  • Content: The actual content of the box, such as text or images.
  • Padding: Space between the content and the border.
  • Border: A line surrounding the padding (if any) and content.
  • Margin: Space outside the border, separating the box from other elements.
